# Background Assets

## Typical role
Large-area visual base for one or more pages.

## Identify by
- large dimensions
- low interaction detail
- environment / scene composition

## Rules
- backgrounds rarely need to be fully reused across all pages
- use them to establish mood, not to carry interaction
- if multiple backgrounds conflict, prefer consistency over raw variety

## Stretching
Backgrounds may be cropped or scaled, but avoid distorting focal art.
