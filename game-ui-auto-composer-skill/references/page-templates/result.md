# Result Page Template

## Purpose
- summarize outcome
- show score / rewards / rank
- offer retry / next / home

## Typical regions
- summary title
- score block
- reward block
- CTA group
- optional character or badge highlight

## Required outputs
- main summary focus
- reward hierarchy
- CTA priority
- motion notes
- implementation notes

## Common components
- Root
- ResultTitle
- ScorePanel
- RewardPanel
- CTAGroup
- OptionalCharacterSlot
