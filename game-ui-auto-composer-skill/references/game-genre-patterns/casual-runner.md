# Casual Runner Pattern

## Typical properties
- pace is medium to fast
- gameplay lane or track occupies the main central view
- HUD should be light
- buttons should not crowd the run view

## Default page set
- Home
- Gameplay HUD
- Result
- optional Mode Select

## UI priorities
- large CTA on Home
- minimal HUD during gameplay
- reward feedback on Result

## Safe-zone rule
Keep central running area clear. Prefer HUD on top edge and corners.

## Good motion defaults
- start button pulse
- score increment
- result reward reveal
