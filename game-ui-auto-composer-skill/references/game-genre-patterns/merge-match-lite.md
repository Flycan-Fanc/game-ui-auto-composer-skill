# Merge / Match Lite Pattern

## Typical properties
- board area or merge field is central
- top bar often shows currency / progress / goals
- bottom area may hold utility or inventory

## Default page set
- Home
- Gameplay HUD
- Result
- optional Level Select

## Safe-zone rule
Keep board area visually clear. Panels should stay on margins.

## Composition tips
- use panels for goals and resource bars
- keep ornament density moderate
- reserve top-left/top-right for status blocks
