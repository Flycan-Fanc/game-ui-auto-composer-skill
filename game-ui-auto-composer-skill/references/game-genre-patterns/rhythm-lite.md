# Rhythm Lite Pattern

## Typical properties
- timing lane or judgement line is sacred
- UI should stay peripheral
- motion is acceptable but must not distract from timing

## Default page set
- Home
- Gameplay HUD
- Result

## Safe-zone rule
Avoid placing dense UI near judgement timing zones.

## Motion defaults
- restrained pulse / feedback
- clear state changes
- avoid excessive screen-wide motion
