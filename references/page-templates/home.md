# Home Page Template

## Purpose
- introduce the game
- establish theme quickly
- expose the primary CTA
- optionally expose mode select / ranking / settings

## Typical regions
- background
- title group
- main CTA
- optional secondary action group
- optional character showcase
- top utility corner

## Required outputs
- main visual focus
- start action
- style note
- asset-role assignment
- component tree
- anchor hints

- Root
- Background
- TitleGroup
- CharacterShowcase
