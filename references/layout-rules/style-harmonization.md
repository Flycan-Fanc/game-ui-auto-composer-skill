# Style Harmonization

## Goal
Produce one coherent interface language from mixed assets.

## Harmonize
- color emphasis
- glow intensity
- rounded vs sharp language
- title treatment
- panel treatment
- button treatment
- decoration density

## Priority order
1. clarity
2. consistency
3. implementation simplicity
4. asset usage coverage

## Rule
Do not use all assets by default. Use only what strengthens the page.
