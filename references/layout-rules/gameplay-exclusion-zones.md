# Gameplay Exclusion Zones

## Purpose
Prevent UI from blocking gameplay-critical space.

## Common patterns
### Runner / lane-switch
Protect the central track or lane field.

### Merge / board puzzle
Protect the board area.

### Rhythm
Protect the timing line and nearby visual focus band.

### Shooter
Protect the combat view center and aiming path.

## Rule
When uncertain, prefer smaller HUD and push interface elements to the perimeter.
