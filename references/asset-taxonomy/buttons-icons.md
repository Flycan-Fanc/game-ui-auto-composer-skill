# Button and Icon Assets

## Button base
Likely used for:
- Home primary CTA
- Result action buttons
- Secondary utility actions

## Icon
Likely used for:
- currency
- settings
- ranking
- hints
- state indicators

## Rules
- keep button language consistent across pages
- icons should not replace readable labels when clarity would suffer
- prefer fewer, clearer buttons over many tiny ones

## States
Interactive buttons should support:
- idle
- pressed
- disabled
- selected if needed
