# Motion Levels

## Level 0: Static
No motion.

## Level 1: Micro
Default for v1.
Examples:
- button press scale
- icon pulse
- popup fade/slide
- reward count-up

## Level 2: Stateful
Use when worthwhile.
Examples:
- HUD warning state
- combo / streak prompts
- danger pulse
- staged reward reveal

## Level 3: Rich
Not default for v1.
Examples:
- large homepage character performance
- complex page transitions
- cinematic result flow
